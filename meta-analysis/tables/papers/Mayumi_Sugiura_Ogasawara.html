Group C: Exposure & Control Definition
RM thresholdDose-response: 0, 1, 2, and ≥3 previous pregnancy losses (PL); no single RM cutoff
Type of lossesAll pregnancy losses reported by participants (self-report + doctor records); includes miscarriage, stillbirth, induced abortion distinguished in records. RPL defined as ≥2 losses at any time during pregnancy.
Gestational age cutoffLosses at any time during pregnancy; most before 12 weeks. Late miscarriage/stillbirth (>20 wks) and early miscarriage (<12 wks) analyzed as separate outcomes in the current pregnancy.
Dose-responseYes — four tiers (0, 1, 2, ≥3 prior losses) for all outcomes; OR reported for each tier vs. 0-loss reference
Control groupWomen with no previous pregnancy loss (n = 73,413), same JECS cohort
Group D: Outcomes & Methods
Outcomes measuredLate miscarriage/stillbirth >20 wks; early miscarriage <12 wks; pregnancy loss (any); PTB <37 wks; PTB <34 wks; PROM; placenta praevia; oligohydramnios; abruptio placentae; placental adhesion; HDP mild; HDP severe; uterine infection; caesarean section; SFD (<10th percentile); IUFD; low birth weight (<2,500 g); infant sex (male); Apgar <7 at 5 min; umbilical artery pH <7.1; congenital anomaly; aneuploidy
Effect measureOR (odds ratio) with 95% CI
Statistical methodLogistic regression with propensity score; four models: Model I (unadjusted), Model II (adjusted, complete-case), Model III (adjusted + MIA), Model IV (MIA + gynecological history). Primary results: Model III.
Confounders adjustedOutcome-specific propensity scores (P < .05 in univariable); variables considered: maternal age, BMI, marital status, IVF-ET, previous live birth, smoking, education, income, and 30+ medical/gynecological history covariates
Group E: Results (primary effect estimates from Model III = MIA + propensity score)

Group F: Quality Notes
Data source qualityHigh — large-scale nationwide birth cohort (n > 96,000); prospective collection; multiple data sources (self-report + medical records); government-funded with standardized protocols
Confounder adjustmentYes — extensive; outcome-specific propensity scores with up to 30+ covariates; four models with increasing adjustment rigor; MIA used to handle 5.2%–21.1% missing data
Outcome validationMedical record transcription by doctors; congenital anomaly from physical examination and neonatal discharge records; chromosomal aneuploidy from doctor records
Bias acknowledgedGeneralizability concerns addressed via MIA; increased surveillance bias (women with PL may have more gynecological diagnoses, hence Model IV); no information on induced abortion in some records
Other notesUse of MIA is a key methodological distinction; some outcomes (PTB <37 wks, uterine infection) became significant only in Model III, illustrating impact of missing data handling. Uses "pregnancy loss" (PL) terminology, including losses at any gestational age. Dose-response mirrors Gunnarsdottir 2014 but in a Japanese population.
Abbreviations: RM = recurrent miscarriage; mc = miscarriage; PL = pregnancy loss; OR = odds ratio; aOR = adjusted odds ratio; CI = confidence interval; NR = not reported; PTB = preterm birth; SFD = small for date; IUFD = intrauterine fetal death; LBW = low birth weight; HDP = hypertensive disorders of pregnancy; PROM = premature rupture of membranes; IVF-ET = in vitro fertilization and embryo transfer; MIA = multiple imputation analyses; JECS = Japan Environment and Children's Study.
